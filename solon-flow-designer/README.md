# 说明
Solon flow designer是vue3 + vite + antd + x6开发的流程设计器。

目的是为了提供一个简单易用的流程设计器，支持流程的设计、导入、导出等功能。

# 安装和运行

1、安装node，访问Node.js 中文网 (nodejs.com.cn)

安装完成后，可以通过命令行输入以下命令查看版本号
```bash
node -v
npm -v
```

2、执行npm install
根据package.json中的依赖版本下载依赖包

```bash
npm install
```

3、运行

```bash
npm run dev
```
